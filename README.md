# 📊 資産運用シミュレーター

初期資産・毎月積立額・想定リターンを自由に設定して、将来の資産推移を予測するツールです。

## 機能

- **3シナリオ比較**（保守的・基本・楽観的）
- **自由な条件設定** — ファンドの追加・削除、保有額・積立額・リターン率をすべて編集可能
- **マイルストーン予測** — 500万、1000万、5000万、1億円の到達時期を表示
- **年次推移表** — 各年の総資産・運用益・利益率を一覧表示

## 想定リターンのデフォルト値

Goldman Sachs, Vanguard, JPMorgan, BlackRock, Research Affiliates, Morningstar等の10年期待リターン予測に基づいています。

## 開発

```bash
npm install
npm run dev
```
